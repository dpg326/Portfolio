Projects

Organoid Sorting via Image Machine Learning

This project was developed during the Mountaintop Summer Experience in June 2024 as an interdisciplinary project within Lehigh’s Bioengineering Department under Professor Yaling Liu. All the code and imaging is under lab confidentiality, but I'm proud to say our research won first prize at Lehigh's 2024 Summer Expo.

Technologies Used: Python, TensorFlow, Keras, OpenCV, Flask, Matplotlib, Seaborn, EfficientNetV2, Pandas, PySide6

We trained a supervised model on 1800+ brightfield organoid images using EfficientNetV2, achieving accuracy rates of around 92%—comparable to our unsupervised model.

Created two UIs: A simple automatic single-organoid image classifier using Flask and Bootstrap, and a more robust tool for classifying clusters of organoids via object detection on static images and live video.
